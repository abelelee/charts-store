# Hasty Paste II
快速粘贴一切内容。

文档地址：[hastypaste.docs.enchantedcode.co.uk](https://hastypaste.docs.enchantedcode.co.uk/)

## 展示
![展示图片，显示多个应用页面](./docs/static/showcase.webp)

## 功能特性
- 粘贴内容可见性
  - 私有
  - 非公开
  - 公开
- 粘贴资源
- 粘贴内容渲染（纯文本、Markdown、代码）
- 自定义粘贴标识（slug）（或随机生成）
- 粘贴内容过期时间
- 用户账户支持“匿名”模式
- 通过 OpenID/OAuth2 的单点登录（SSO）
- 暗色/亮色主题
