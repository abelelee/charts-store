一个免费且开源的机器翻译 API，完全支持自托管。与其他 API 不同，它不依赖 Google 或 Azure 等专有提供商进行翻译。相反，它的翻译引擎由开源的 [Argos Translate](https://github.com/argosopentech/argos-translate) 库驱动。
